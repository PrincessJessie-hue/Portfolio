Calmora: SaaS Landing Page Design

Calmora is a mental wellness SaaS concept designed to help users manage stress and emotional wellbeing through guided tools and digital support. This landing page was designed and built to communicate trust, calmness, and clarity.

View Live Website
Design Highlights

Soft UI

The interface uses rounded shapes, breathing space, and soft colors to create a calming visual experience aligned with the emotional goals of the platform.


Clear Visual Hierarchy

Information is structured so that users can quickly understand the benefits of the product without feeling overwhelmed.

Key Features

Conversion-Focused Layout

The layout guides visitors through the product story, gradually building trust before presenting the call-to-action.


Emotional Design

Visual elements were designed to evoke calmness and safety, which are essential for a mental wellness platform.


Strategic CTAs

Calls to action are positioned throughout the page so users can engage with the product at multiple moments of interest.

Process

The project began with wireframes in Figma to establish the layout and content structure. Once the visual direction was defined, the design was translated into a responsive web layout using HTML and CSS.
