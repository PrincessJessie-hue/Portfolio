KHY: A Visual Website Redesign

This project explores an alternative visual direction for the KHY fashion website. The goal was to experiment with a more editorial, fashion-forward layout that emphasizes storytelling and visual impact.

Current Website

The existing KHY website uses a traditional e-commerce layout focused on product grids and minimal editorial presentation.

New Vision

The redesign introduces a more immersive layout inspired by digital fashion editorials. Large imagery and dramatic spacing allow the collection to feel more expressive and premium.

Typography & Palette

The redesign emphasizes bold typography paired with neutral tones to create a high-fashion aesthetic.

Key Features

Editorial Layout

The layout focuses on large visuals and dramatic spacing to create a fashion magazine style experience.


Stronger Visual Hierarchy

Typography and imagery were restructured to guide the viewer's attention more intentionally.


Immersive Storytelling

The redesign presents the collection as a narrative rather than simply a product grid.
